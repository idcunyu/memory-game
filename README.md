## 游戏介绍

本项目是一款考验记忆小游戏，简单耐玩。

游戏中总共有8组分散的卡片，共16张，每局游戏开始或restart之后，卡片位置洗牌（每一局都不一样）；当开始点击卡片时，视为开始本局游戏，开始计时、计步数（2次点击为1步）；需要注意的是，影响最终成绩（星星数量）的只有步数，计时仅做参考。

当两次点开的卡片不匹配时，卡片只显示一会儿，然后翻转隐藏；当点开的卡片匹配后，卡片变为显示状态。

当八组卡片全部匹配后，停止计时、计步，得出本局成绩并弹出成绩页面。


## 得分规则

| :-----:   | :---:  |
